# HCMS Test Automation

## Overview

This repository demonstrates how manual regression testing for the Healthcare Case Management System (HCMS) can be automated after successful software validation.

The automation scenarios are based on validated test cases from the **software-validation-lifecycle** repository.

---

## Objective

After a software change has been validated and deployed, repetitive regression testing is automated to improve efficiency, consistency, and execution speed.

---

## Automation Workflow

Software Validation

↓

Production Release

↓

↓

Automation Script Developed

↓

Scheduled Execution

↓

Execution Report Generated
